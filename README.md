# browser template

webpack + Babel + React + Pug + SCSS

## Requirements

### Compile pug files

Use `pug-cli`

```
npm i pug-cli -g
```

## Scripts

* `npm run pug`: Compile `.pug`
* `npm run build`: Compile `.jsx` -> `.js` & `.scss` -> `.css`
