# browser template

webpack + Babel + React + Pug + SCSS

## Requirements

### Compile pug files

Use `pug-cli`

```
npm i pug-cli -g
```

## Scripts

* `npm run pug`: Compile `.pug`
* `npm run build`: `webpack`
* `npm run build-with-source-map`: `webpack -d`
* `npm run watch`: Run `webpack-dev-server`
